# คู่มือใช้งาน Pet Focus

แอปช่วยลดเวลาใช้มือถือสำหรับนักเรียน โดยใช้สัตว์เลี้ยงเสมือนเป็นแรงจูงใจ — โฟกัสให้สำเร็จ สัตว์เลี้ยงมีความสุข ได้เหรียญไปแต่งตัวและตกแต่งห้อง

**รองรับ Android เท่านั้น**

---

## 1. ติดตั้งและลงทะเบียน

### ดาวน์โหลดและติดตั้ง

1. ดาวน์โหลดไฟล์ APK จากหน้าเว็บของโรงเรียน (หน้า **ดาวน์โหลด**)
2. เปิดไฟล์ APK บนมือถือ Android
3. ถ้าระบบถาม ให้อนุญาต **Install unknown apps** สำหรับเบราว์เซอร์หรือ File Manager ที่ใช้เปิดไฟล์
4. ติดตั้งเสร็จแล้วเปิดแอป **Pet Focus**

### ลงทะเบียนครั้งแรก

1. เปิดแอปครั้งแรกจะเห็นหน้า **ลงทะเบียน**
2. ใส่ **รหัสนักเรียน** ที่ได้รับจากโรงเรียน
3. กด **เริ่มใช้งาน**

> **สำคัญ:** 1 รหัสนักเรียน = 1 เครื่องเท่านั้น  
> ถ้าลบแอปแล้วติดตั้งใหม่บนเครื่องเดิม ใช้รหัสเดิมได้ตามปกติ

---

## 2. ภาพรวมหน้าจอหลัก

แอปมี 5 แท็บด้านล่าง:

| แท็บ | หน้าที่ |
|------|---------|
| 🐱 **หน้าหลัก** | ดูสัตว์เลี้ยง สถานะ และของตกแต่งในห้อง |
| ⏱ **โฟกัส** | เริ่ม Focus session วางมือถือ |
| ✅ **เควส** | ทำงานบ้านถ่ายรูปรับเหรียญ |
| 📊 **สถิติ** | ดู screen time และสรุปสัปดาห์ |
| 🛍️ **ร้านค้า** | ซื้อของแต่งตัว/ตกแต่งห้อง |

---

## 3. หน้าหลัก (Home)

- **สัตว์เลี้ยง** แสดงแอนิเมชันตามสายพันธุ์ (แมว/หมา)
- **แตะชื่อสัตว์เลี้ยง** เพื่อเปลี่ยนชื่อ (1–12 ตัวอักษร)
- **ปุ่มลอยด้านขวา** สลับระหว่างสัตว์เลี้ยง (หมาต้องซื้อจากร้านค้าก่อน)
- แสดง **HP, ความสุข, Level, เหรียญ ⭐** ของสัตว์เลี้ยง

---

## 4. โหมดโฟกัส (Focus Mode)

### วิธีใช้

1. ไปที่แท็บ **โฟกัส**
2. เลือกเวลา: **30 / 60 / 120 / 180 นาที**
3. กด **เริ่มโฟกัส**
4. วางมือถือไว้ ไม่เปิดแอปที่ห้ามใช้
5. เมื่อครบเวลา จะได้รางวัลและบันทึกอารมณ์ (Mood check-in)

### รางวัลเมื่อสำเร็จ

| เวลา | เหรียญ ⭐ | EXP |
|------|----------|-----|
| 30 นาที | 10 | 20 |
| 1 ชั่วโมง | 25 | 50 |
| 2 ชั่วโมง | 60 | 120 |
| 3 ชั่วโมง | 100 | 200 |

+ ความสุขสัตว์เลี้ยง +10

### แอปที่ห้ามเปิดระหว่างโฟกัส

ถ้าเปิดแอปเหล่านี้ระหว่าง session จะ **ล้มเหลว**:

- TikTok
- Instagram
- Facebook
- Netflix
- YouTube

> เปิดแอปอื่นที่ไม่อยู่ในรายการ / ปิดหน้าจอ / อยู่หน้า Home ของ Pet Focus → **นับเวลาต่อได้**

### สิ่งที่ต้องอนุญาต

ครั้งแรกที่โฟกัส แอปจะขอสิทธิ์:

- **Usage Access** — ตรวจว่าเปิดแอปไหนอยู่ (จำเป็นสำหรับตรวจแอปที่ห้าม)
- **การแจ้งเตือน** — แสดงเวลานับถอยหลังขณะแอปอยู่เบื้องหลัง

เปิดสิทธิ์ได้จาก Settings ของ Android หรือจากหน้า **สถิติ**

### ถ้าโฟกัสล้มเหลว

- Session หยุดทันที
- **ไม่มีโทษ** — ไม่หักเหรียญหรือ HP
- ลองใหม่ได้เมื่อพร้อม

---

## 5. เควสทำงานบ้าน

เควสรายวัน 4 อย่าง แต่ละเควสได้ **15 เหรียญ ⭐**:

| เควส | วิธีถ่ายรูป |
|------|-------------|
| กวาดบ้าน | ให้เห็นทั้งใบหน้าและบริบทในบ้าน/พื้น |
| ล้างจาน | ให้เห็นทั้งใบหน้าและอ่าง/จาน |
| จัดโต๊ะเรียน | ให้เห็นทั้งใบหน้าและโต๊ะที่จัดแล้ว |
| รีดเสื้อผ้า | ให้เห็นทั้งใบหน้าและเสื้อผ้า/เตารีด |

### วิธีทำเควส

1. ไปที่แท็บ **เควส**
2. **ลงทะเบียนใบหน้าก่อน** (ครั้งแรกเท่านั้น — ดูหัวข้อ 7)
3. เลือกเควส → **ถ่ายรูปในแอป** หรือ **เลือกจากคลัง**
4. ระบบตรวจ **ใบหน้า + งานบ้านในภาพ** บนเครื่อง (ไม่ส่งรูปขึ้น cloud)
5. ผ่านแล้วได้เหรียญ — ลองใหม่ได้ไม่จำกัดจนกว่าจะสำเร็จ

> รูปจากคลังต้องถ่าย **วันนี้** เท่านั้น

---

## 6. ร้านค้า

ใช้เหรียญ ⭐ ที่สะสมได้ซื้อของ:

- **แต่งตัวสัตว์เลี้ยง** — หมวก, แว่น ฯลฯ
- **ตกแต่งห้อง** — พื้นหลัง, ของวางในห้อง
- **สัตว์เลี้ยง** — ซื้อ **หมา** 500 เหรียญ (แมวมีให้ตั้งแต่เริ่มต้น)

ของที่ซื้อแล้วสามารถ **สวม/วาง/ตั้ง theme** ได้จากร้านค้า

---

## 7. ลงทะเบียนใบหน้า

- **ไม่บังคับ** ตอนเปิดแอปครั้งแรก
- **จำเป็น** ก่อนทำเควสทำงานบ้าน
- เปิดได้จากหน้า **เควส** หรือ **สถิติ**
- เก็บข้อมูลใบหน้า **บนเครื่องเท่านั้น** (ไม่เก็บรูป ไม่ sync ขึ้น server)
- ลงทะเบียนใหม่ได้จากหน้า **สถิติ**

**เคล็ดลับ:** ให้ใบหน้าใหญ่พอในภาพ (~25% ความกว้าง) ตอนลงทะเบียน

---

## 8. สถิติ (Stats)

หน้านี้แสดง:

- **Screen time วันนี้** — เวลาใช้มือถือรวม
- **Streak** — จำนวนวันติดต่อที่โฟกัสสำเร็จ
- **สรุปสัปดาห์** — session, นาทีโฟกัส, อารมณ์เฉลี่ย
- **กราฟแอป** — แอปไหนใช้เวลามากที่สุด
- **AI Tip** — คำแนะนำปรับพฤติกรรม
- **ลงทะเบียนใบหน้าใหม่**
- เปิด **Settings สิทธิ์** (Usage Access, การแจ้งเตือน)

## 10. แก้ปัญหาเบื้องต้น

| อาการ | วิธีแก้ |
|--------|---------|
| โฟกัสไม่ตรวจจับแอปที่ห้าม | เปิด **Usage Access** ให้ Pet Focus ใน Settings |
| Timer ไม่เดินตอนปิดแอป | อนุญาต **การแจ้งเตือน** ให้แอป |
| เควสไม่ผ่าน | ให้เห็นทั้งใบหน้าและงานบ้านชัดในภาพ ลองถ่ายใหม่ |
| รหัสนักเรียนใช้ไม่ได้ | ตรวจว่าไม่ได้ใช้บนเครื่องอื่นแล้ว — ติดต่อครู/ผู้ดูแล |
| แอปค้างหน้า loading | รอสักครู่ หรือล้างข้อมูลแอป (Settings → Apps → Pet Focus → Clear data) แล้วลงทะเบียนใหม่ |

---

## 11. สำหรับครู/ผู้ดูแล (Admin Web)

เว็บ Admin ใช้ดูข้อมูลนักเรียน:

- Dashboard สรุปจำนวนนักเรียน / focus วันนี้ / screen time
- รายชื่อนักเรียน → ดูรายละเอียด focus, screen time 7 วัน, เควส

เข้าสู่ระบบด้วย **username + password** ที่ได้รับจากผู้ดูแลระบบ

## คำถามที่พบบ่อย

**Q: ลบแอปแล้วติดตั้งใหม่ ใช้รหัสเดิมได้ไหม?**  
A: ได้ ถ้าเป็นเครื่องเดิม ระบบจะ restore session ให้

**Q: โฟกัสล้มเหลวแล้วโดนลงโทษไหม?**  
A: ไม่ — แอปใช้แรงจูงใจเชิงบวก ไม่หักเหรียญหรือ HP

**Q: รูปเควสถูกส่งไป server ไหม?**  
A: ไม่ — ตรวจบนเครื่องเท่านั้น ไม่เก็บรูป

**Q: ใช้ iPhone ได้ไหม?**  
A: ยังไม่รองรับ — ใช้ได้เฉพาะ Android
